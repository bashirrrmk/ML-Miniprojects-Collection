# 🤖 ML Miniprojects Collection

A collection of machine learning projects built with Python and scikit-learn.

---

## 📁 Projects

### 🔬 1. Breast Cancer Classification — Random Forest
Classifies breast tumors as **Malignant** or **Benign** using the Wisconsin Diagnostic Dataset.

- **Dataset:** 569 samples · 30 features · `sklearn.datasets`
- **Model:** RandomForestClassifier (200 trees) + StandardScaler + GridSearchCV
- **Results:** 95.6% Accuracy · 97.2% Recall · ~0.99 AUC-ROC
- **Dashboard:** Interactive Streamlit app with EDA, ROC curve, feature importances & live prediction

---

### 📉 2. Regression vs Classification
Comparison of Linear Regression and Logistic Regression on real-world datasets.

- **Datasets:** Student Performance · Titanic
- **Models:** Linear Regression · Logistic Regression
- **Goal:** Understand the difference between regression and classification tasks
